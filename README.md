# 📚 ERPNext Library Management System

## 📌 Overview

This project is a Library Management System built using ERPNext as part of an entry-level developer assignment.

---

## 🚀 Features

### 👤 Library Member

* Add and manage members
* Store contact details

### 📖 Book Management

* Add books with details (Author, ISBN)
* Track availability status

### 🔄 Book Transactions

* Issue books to members
* Return books
* Automatic availability update

---

## ⚙️ Functionality

* Book becomes unavailable when issued
* Book becomes available when returned
* Linked DocTypes for proper data flow

---

## 🧪 Testing

* Member creation tested
* Book creation tested
* Issue and return flow verified
* Validation for unavailable books implemented

---

## 🛠 Tech Stack

* ERPNext (v14/v15)
* Frappe Framework
* Python
